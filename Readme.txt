Trends in Police Traffic Stops in the United States

This project aims to analyze the data collected by the STANFORD OPEN POLICING PROJECT

download data from: https://openpolicing.stanford.edu/

Project runs on Spark.  
Please refer the report.pdf for more details

MLtree folder cotains the code to build a decision tree to see if a person is stopped will he be searched/arrested? given his age, gender and state(where stop occurred) 
DataCount: counts the number of people stopped/searched(by age, gender and race) in a particular state.
